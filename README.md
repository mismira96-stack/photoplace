# 사진 자동정리

안드로이드 `DCIM/Camera` 사진을 기존 위치 앨범으로 복사하는 앱 초안입니다.

## 동작 방식

1. `새로 확인`을 눌러 `DCIM/Camera` 사진을 읽습니다.
2. 앱이 GPS 위치와 기존 앨범명을 비교합니다.
3. 기존 앨범에 위치명이 있으면 그 앨범으로 복사합니다.
4. 기존 앨범이 없으면 `{지역명}에서` 폴더를 만들고 복사합니다.
5. 위치 정보가 없는 사진은 이동/복사하지 않고 건너뜁니다.
6. 복사 결과를 확인한 뒤 `복사된 카메라 원본 삭제` 버튼으로 원본 삭제를 따로 요청할 수 있습니다.
7. `갤러리에서 확인` 버튼으로 갤러리 앱을 열 수 있습니다.

## 생성되는 폴더

```text
Pictures/
  안동에서/
    IMG_0001.jpg
  청주에서/
    IMG_0003.jpg
```

날짜별 하위 폴더를 만들지 않습니다.
파일명도 바꾸지 않습니다.

## 중복 기준

현재 버전은 대상 폴더에 같은 파일명이 있으면 건너뜁니다.

## 빌드

Android Studio에서 이 폴더를 열고 `Run`을 누르면 됩니다.

터미널에서는 아래 명령으로 디버그 APK를 만들 수 있습니다.

```powershell
.\gradlew.bat assembleDebug
```

빌드 결과:

```text
app/build/outputs/apk/debug/app-debug.apk
```
