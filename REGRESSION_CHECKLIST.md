## 출시 전 최종 회귀

- [ ] 앱 신규 설치 후 홈 / 정리기록 / 설정 3탭 표시
- [ ] 분석할 폴더 선택에서 원본 폴더만 표시
- [ ] 위치 앨범 폴더(`수원에서`, `송파구에서` 등)는 분석 폴더 목록에서 제외
- [ ] 사진 위치 인식 후 위치별 앨범 생성
- [ ] 동영상 위치 인식 후 위치별 앨범으로 이동
- [ ] 위치 없는 항목은 위치 앨범으로 이동하지 않음
- [ ] 사진 원본은 복사 후 휴지통 이동 가능
- [ ] 동영상은 정리 시 이동되며 별도 휴지통 대상에 포함되지 않음
- [ ] 오래된 사진이 Instagram / KakaoTalk / Google Photos 최근 항목으로 튀지 않음
- [ ] 정리 완료 후 정리기록 목록에 새 장소 또는 기존 장소 갱신 반영
- [ ] 정리기록 상세 진입 후 시스템 Back / 상단 Back으로 정리기록 목록 복귀
- [ ] 삼성 갤러리에서 해당 앨범으로 직접 이동
- [ ] 삼성 갤러리 앨범이 비었거나 삭제된 경우 앱 크래시 없이 목록 갱신
- [ ] Google Drive 등 외부 백업 후 로컬 파일이 사라진 항목은 정리기록/홈 카드에서 제외 또는 갱신
- [ ] 기억 한 줄 추가 / 수정 / 삭제 후 앱 재실행해도 유지
- [ ] 장소 별칭 또는 기억이 장소명과 분리되어 표시
- [ ] 대표 썸네일 로딩 실패 시 크래시 없음
- [ ] 날짜 범위가 정리기록 목록과 상세 화면에서 자연스럽게 표시
- [ ] 분석 중 탭 이동 시 백그라운드 진행 배너 표시
- [ ] 앱 강제 종료 후 재실행 시 저장된 정리기록 로드

## 특히 다시 볼 위험 구간

- 원본 사진/동영상의 `DATE_ADDED`, `DATE_MODIFIED`, `DATE_TAKEN` 변경 여부
- KakaoTalk / Screenshots / Download 폴더 포함 분석
- 기존 위치 앨범에 새 항목이 추가될 때 기존 정리기록과 병합되는지
- 삼성 갤러리의 "모든 앨범 보기"가 꺼져 있을 때 사용자 안내 필요 여부
